JAVASCRIPT TYPES
-----------------
1. Number (u can use operators on them. NaN's type is also a number -> Not A Number)
2. String (text -> between " or '. use + to concatenate 2 or more strings. Use \ to override character meanings)
3. Boolean (true or false)
4. Undefined (a variable that hasn't been assigned a value)
5. Null
<!-- 6. Symbol (new in ECMAScript 6) -->
7. Object
